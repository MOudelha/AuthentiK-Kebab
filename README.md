- `assets/css/style.css`
- `assets/js/script.js`
- `assets/img/*`

## Mettre en ligne sur GitHub Pages

1. Crée un dépôt GitHub.
2. Dépose **tout le contenu** de ce dossier à la racine du dépôt.
3. Va dans **Settings > Pages**.
4. Choisis **Deploy from a branch**.
5. Sélectionne la branche `main` et le dossier `/root`.
6. Enregistre : ton site sera publié en quelques instants.

## À personnaliser

- Bouton de commande / livraison

## Remarque importante

Certaines lignes du PDF original sont difficiles à lire. Dans le site, elles sont affichées comme **"Prix à confirmer"** pour rester fidèle au document source.
